# لوحة الإعلانات - Flask App

موقع بسيط مبني بـ Python + Flask يحتوي على:
- صفحة رئيسية تعرض الإعلانات للجميع
- صفحة تسجيل دخول خاصة بالأدمن
- لوحة تحكم (Admin) تسمح بإضافة/تعديل/حذف الإعلانات (البيانات) مباشرة

## طريقة التشغيل

1. تثبيت Flask:
```
pip install flask
```

2. تشغيل التطبيق:
```
python app.py
```

3. افتح المتصفح على:
```
http://127.0.0.1:5000
```

## بيانات تسجيل دخول الأدمن (تجريبية)
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

يمكنك تغييرهما في أعلى ملف `app.py` (المتغيرات ADMIN_USERNAME و ADMIN_PASSWORD).

## ملاحظات
- البيانات تُخزَّن في قاعدة بيانات SQLite تُنشأ تلقائياً باسم `data.db` عند أول تشغيل.
- هذا مثال تعليمي بسيط؛ في مشروع حقيقي يفضل تشفير كلمة المرور (مثلاً باستخدام werkzeug.security) واستخدام نظام مستخدمين حقيقي بدل القيم الثابتة.
